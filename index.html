a starter kit for personal automation tools, built on firebase.

prerequisites
  • a recent macos
  • a google account (we'll set up a dedicated one below)
getting started
  1. create a dedicated gmail account for this project
  2. create a github account, signing in with that gmail
  3. request access ↓ (we'll review and add you as a collaborator)
✓ got it. you'll get an email from github when you've been added.
what happens next

once approved, github emails you an invite to the repo. accept it, open the repo, and follow the readme.

while you wait, you can
  • enable google cloud free trial on your dedicated gmail (console.cloud.google.com/freetrial)
  • create a dedicated macos user account on the machine you'll run if from
